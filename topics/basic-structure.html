Basic HTML Structure

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements and tags.

Basic Document Structure

Every HTML document has the same basic structure:

Example: Basic HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph of text.</p>
</body>
</html>

Key Components

  • <!DOCTYPE html> - Declares the document type as HTML5
  • <html> - Root element that contains all content
  • <head> - Contains metadata about the document
  • <body> - Contains the visible content of the page

HTML Elements and Tags

HTML elements are the building blocks of HTML pages:

  • Opening tag: <tagname>
  • Content: The text or other elements inside
  • Closing tag: </tagname>

HTML Attributes

Attributes provide additional information about elements:

Example: Attributes

<img src="image.jpg" alt="Description" width="300">
<a href="https://example.com" target="_blank">Link</a>
<p class="highlight" id="intro">Text</p>

Self-Closing Tags

Some elements don't need closing tags:

  • <img> - Images
  • <br> - Line breaks
  • <hr> - Horizontal rules
  • <input> - Form inputs

Comments

Use comments to add notes in your code:

Example: HTML Comments

<!-- This is a comment -->
<p>This paragraph will be visible</p>
<!--
    Multi-line
    comment
-->
