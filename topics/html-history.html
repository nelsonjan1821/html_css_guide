The History of HTML

The Beginning (1989-1991)

Tim Berners-Lee invented HTML in 1989 while working at CERN (European Organization for Nuclear Research) in Switzerland. He created it as part of his vision for the World Wide Web.

Original Goal

Berners-Lee wanted to create a system where scientists could easily share documents and research across different computer systems worldwide.

HTML Timeline

HTML 1.0 (1991)

  • First public description of HTML
  • Basic elements: headings, paragraphs, lists, links
  • Only 18 elements total
  • No images, forms, or styling

Example: Original HTML (1991)

<TITLE>Simple Document</TITLE>
<H1>Main Heading</H1>
<P>This is a paragraph with a <A HREF="page2.html">link</A>.
<UL>
<LI>First item
<LI>Second item
</UL>

HTML 2.0 (1995)

  • First official HTML standard (RFC 1866)
  • Added forms and file uploads
  • Introduced image support
  • Tables were added later

HTML 3.2 (1997)

  • W3C (World Wide Web Consortium) took over HTML development
  • Added tables, applets, text flow around images
  • Math formulas support
  • Style sheets were beginning to emerge

HTML 4.0 (1997) and 4.01 (1999)

  • Introduced CSS (Cascading Style Sheets)
  • Added accessibility features
  • Frames for page layout
  • Scripting support (JavaScript)
  • Internationalization support

Example: HTML 4.0 Features

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN">
<HTML>
<HEAD>
    <TITLE>HTML 4.0 Document</TITLE>
    <LINK REL="stylesheet" TYPE="text/css" HREF="style.css">
</HEAD>
<BODY>
    <TABLE BORDER="1">
        <TR><TD>Cell 1</TD><TD>Cell 2</TD></TR>
    </TABLE>
</BODY>
</HTML>

XHTML 1.0 (2000)

  • HTML reformulated as XML application
  • Stricter syntax rules
  • All tags must be lowercase
  • All tags must be properly closed
  • All attributes must be quoted

Example: XHTML Syntax

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <title>XHTML Document</title>
</head>
<body>
    <p>All tags must be <em>properly closed</em>.</p>
    <img src="image.jpg" alt="Description" />
    <br />
</body>
</html>

HTML5 (2014)

  • Major revision after years of development
  • New semantic elements (header, nav, article, section)
  • Native audio and video support
  • Canvas for graphics
  • Local storage capabilities
  • New form input types
  • Simplified DOCTYPE

Example: HTML5 Features

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>HTML5 Document</title>
</head>
<body>
    <header>
        <nav>Navigation here</nav>
    </header>
    <main>
        <article>
            <video controls>
                <source src="movie.mp4" type="video/mp4">
            </video>
        </article>
    </main>
    <footer>Footer content</footer>
</body>
</html>

Key Innovations Through Time

1991: Hypertext

The revolutionary idea of linking documents together across the internet.

1993: Images

The <img> tag transformed the web from text-only to visual.

1995: Forms

Made the web interactive, allowing user input and data submission.

1997: CSS

Separated content from presentation, enabling better design control.

2005: AJAX

Although not HTML itself, enabled dynamic web applications.

2014: HTML5

Brought multimedia, semantics, and app-like capabilities to the web.

Modern HTML (Present)

HTML continues to evolve with:

  • Living Standard: Continuous updates rather than versioned releases
  • Web Components: Custom elements and shadow DOM
  • Progressive Web Apps: Making web apps feel like native apps
  • Accessibility Focus: Better support for users with disabilities
  • Mobile-First: Designed for mobile devices

Impact on the World

HTML's Global Impact

  • Democratized Publishing: Anyone can create and share content
  • Connected the World: Information accessible globally
  • Economic Revolution: Enabled e-commerce and digital economy
  • Education: Online learning and knowledge sharing
  • Social Connection: Foundation for social media and communication

Fun Facts

  • The first website ever created is still online: http://info.cern.ch/hypertext/WWW/TheProject.html
  • HTML was initially inspired by SGML (Standard Generalized Markup Language)
  • The first web browser was also called "WorldWideWeb" (later renamed Nexus)
  • Tim Berners-Lee wrote the first web server, web browser, and website
  • The <blink> tag was once a real HTML element (thankfully deprecated!)
