Text and Content Elements

Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important):

Example: Headings

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>
<h4>Sub-subsection Title</h4>
<h5>Minor Heading</h5>
<h6>Smallest Heading</h6>

Paragraphs

Use <p> tags to create paragraphs of text:

Example: Paragraphs

<p>This is the first paragraph.</p>
<p>This is the second paragraph with more text.</p>

Line Breaks and Horizontal Rules

  • <br> - Creates a line break
  • <hr> - Creates a horizontal rule (divider line)

Example: Breaks and Rules

<p>First line<br>Second line</p>
<hr>
<p>Content after the horizontal rule</p>

Text Formatting

HTML provides several elements for formatting text:

  • <strong> - Important text (bold)
  • <em> - Emphasized text (italic)
  • <b> - Bold text (visual only)
  • <i> - Italic text (visual only)
  • <u> - Underlined text
  • <s> - Strikethrough text
  • <mark> - Highlighted text
  • <small> - Smaller text
  • <sub> - Subscript
  • <sup> - Superscript

Example: Text Formatting

<p>This is <strong>important</strong> text.</p>
<p>This is <em>emphasized</em> text.</p>
<p>This is <mark>highlighted</mark> text.</p>
<p>H<sub>2</sub>O and E=mc<sup>2</sup></p>

Quotations

  • <blockquote> - Block quotations
  • <q> - Inline quotations
  • <cite> - Citations

Example: Quotations

<blockquote>
    This is a block quote that takes up
    its own paragraph space.
</blockquote>

<p>As Shakespeare said, <q>To be or not to be</q>.</p>

<p>Reference: <cite>Hamlet by William Shakespeare</cite></p>

Preformatted Text

Use <pre> to preserve whitespace and formatting:

Example: Preformatted Text

<pre>
    This    text    preserves
        all    spacing
            and    line breaks
</pre>

Code Elements

  • <code> - Inline code
  • <kbd> - Keyboard input
  • <samp> - Sample output
  • <var> - Variables

Example: Code Elements

<p>Use the <code>console.log()</code> function.</p>
<p>Press <kbd>Ctrl</kbd> + <kbd>C</kbd> to copy.</p>
<p>Output: <samp>Hello World</samp></p>
<p>The variable <var>x</var> equals 5.</p>
