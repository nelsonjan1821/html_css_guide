Images and Media

Basic Images

The <img> element displays images:

Example: Basic Image

<img src="image.jpg" alt="Description of the image">

Image Attributes

  • src - Source URL of the image (required)
  • alt - Alternative text for accessibility (required)
  • width - Width in pixels
  • height - Height in pixels
  • title - Tooltip text
  • loading - Loading behavior (lazy, eager)

Example: Image with Attributes

<img src="photo.jpg"
     alt="Beautiful sunset over mountains"
     width="600"
     height="400"
     title="Sunset Photo"
     loading="lazy">

Responsive Images

Make images adapt to different screen sizes:

Example: Responsive Image

<img src="image-large.jpg"
     srcset="image-small.jpg 480w,
             image-medium.jpg 800w,
             image-large.jpg 1200w"
     sizes="(max-width: 480px) 100vw,
            (max-width: 800px) 50vw,
            25vw"
     alt="Responsive image">

Figure and Caption

Group images with captions using semantic elements:

Example: Figure with Caption

<figure>
    <img src="chart.png" alt="Sales data chart">
    <figcaption>
        Sales increased by 25% in the last quarter
    </figcaption>
</figure>

Audio Elements

Embed audio content with the <audio> element:

Example: Audio Player

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>

Audio Attributes

  • controls - Show playback controls
  • autoplay - Start playing automatically
  • loop - Repeat the audio
  • muted - Start muted
  • preload - How much to preload (none, metadata, auto)

Video Elements

Embed video content with the <video> element:

Example: Video Player

<video width="640" height="360" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">
    Your browser does not support the video element.
</video>

Video Attributes

  • controls - Show playback controls
  • autoplay - Start playing automatically
  • loop - Repeat the video
  • muted - Start muted
  • poster - Image to show before playing
  • preload - How much to preload

Embedded Content

Embed external content using various elements:

Example: Embedded Content

<!-- YouTube video -->
<iframe width="560" height="315"
        src="https://www.youtube.com/embed/VIDEO_ID"
        title="YouTube video player"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen>
</iframe>

<!-- Google Maps -->
<iframe src="https://www.google.com/maps/embed?pb=..."
        width="600" height="450"
        style="border:0;"
        allowfullscreen=""
        loading="lazy">
</iframe>

Object and Embed

Alternative ways to embed content:

Example: Object and Embed

<!-- PDF document -->
<object data="document.pdf" type="application/pdf" width="600" height="400">
    <p>Your browser doesn't support PDFs.
       <a href="document.pdf">Download the PDF</a>.
    </p>
</object>

<!-- Flash content (deprecated) -->
<embed src="animation.swf" width="400" height="300">

Media Best Practices

  • Always include meaningful alt text for images
  • Optimize image file sizes for web performance
  • Use appropriate image formats (JPEG, PNG, WebP, SVG)
  • Provide multiple audio/video formats for browser compatibility
  • Use lazy loading for images below the fold
  • Include fallback content for unsupported media
  • Consider accessibility (captions, audio descriptions)
  • Test media elements across different browsers
