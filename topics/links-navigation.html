Links and Navigation

Basic Links

The <a> (anchor) element creates hyperlinks:

Example: Basic Link

<a href="https://www.example.com">Visit Example.com</a>

Link Attributes

  • href - Specifies the destination URL
  • target - Specifies where to open the link
  • rel - Specifies the relationship between current and linked document
  • title - Provides additional information (tooltip)

External Links

Links to other websites:

Example: External Links

<a href="https://www.google.com" target="_blank" rel="noopener">
    Open Google in new tab
</a>

<a href="https://www.github.com" title="Visit GitHub">
    GitHub
</a>

Internal Links

Links to other pages within your website:

Example: Internal Links

<a href="about.html">About Us</a>
<a href="contact.html">Contact</a>
<a href="../index.html">Home</a>

Anchor Links (Same Page)

Links to sections within the same page:

Example: Anchor Links

<!-- Link to section -->
<a href="#section1">Go to Section 1</a>

<!-- Target section -->
<h2 id="section1">Section 1</h2>

<!-- Back to top -->
<a href="#top">Back to top</a>

Email and Phone Links

Special link types for communication:

Example: Contact Links

<!-- Email link -->
<a href="mailto:someone@example.com">Send Email</a>

<!-- Email with subject and body -->
<a href="mailto:support@example.com?subject=Help&body=I need help with...">
    Contact Support
</a>

<!-- Phone link -->
<a href="tel:+1234567890">Call us: (123) 456-7890</a>

Download Links

Links that trigger file downloads:

Example: Download Links

<a href="document.pdf" download>Download PDF</a>

<a href="image.jpg" download="my-image.jpg">
    Download Image
</a>

Navigation Menus

Organize links into navigation structures:

Example: Navigation Menu

<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="services.html">Services</a></li>
        <li><a href="contact.html">Contact</a></li>
    </ul>
</nav>

Breadcrumb Navigation

Show the user's location in the site hierarchy:

Example: Breadcrumbs

<nav aria-label="Breadcrumb">
    <ol>
        <li><a href="/">Home</a></li>
        <li><a href="/products">Products</a></li>
        <li>Current Page</li>
    </ol>
</nav>

Link Best Practices

  • Use descriptive link text (avoid "click here")
  • Use target="_blank" with rel="noopener" for external links
  • Ensure links have good color contrast
  • Test that all links work correctly
  • Use title attributes for additional context when helpful
  • Make navigation consistent across your site
