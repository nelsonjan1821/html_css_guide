Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and visual appearance of HTML documents. While HTML provides the structure and content, CSS controls how that content looks and is laid out on the page.

CSS Stands For:

  • Cascading: Styles can be inherited and overridden in a specific order
  • Style: Controls the visual presentation
  • Sheets: Collections of style rules

Why Do We Need CSS?

Before CSS, web pages were styled using HTML attributes and elements, which made code messy and hard to maintain. CSS separates content from presentation, offering numerous benefits:

Benefits of CSS

  • Separation of Concerns: Content (HTML) separate from presentation (CSS)
  • Consistency: Apply the same styles across multiple pages
  • Maintainability: Change the look of an entire website by editing one CSS file
  • Flexibility: Create responsive designs for different devices
  • Performance: Faster loading times and smaller file sizes
  • Accessibility: Better support for screen readers and assistive technologies

Brief History of CSS

1994

CSS Concept: H√•kon Wium Lie proposes the first CSS specification while working with Tim Berners-Lee at CERN.

1996

CSS1: First CSS specification released by W3C. Included basic properties for fonts, colors, and layout.

1998

CSS2: Added support for media types, positioning, and more selectors. Introduced concepts for print stylesheets.

1999

CSS2.1: Refined CSS2 with bug fixes and better browser compatibility. Became the stable standard for many years.

2011

CSS3: Modular approach with new features like animations, gradients, flexbox, and grid layout. Still evolving today.

Present

Modern CSS: Continuous evolution with new features like CSS Grid, Custom Properties, and Container Queries.

What Can CSS Do?

CSS Controls:

  • Colors & Backgrounds: Text colors, background colors, gradients, images
  • Typography: Fonts, sizes, spacing, line height, text decoration
  • Layout: Positioning, spacing, alignment, responsive design
  • Size & Spacing: Width, height, margins, padding, borders
  • Visual Effects: Shadows, transitions, animations, transforms
  • Responsive Design: Different styles for different screen sizes

CSS Syntax Preview

CSS uses a simple syntax consisting of selectors, properties, and values:

Basic CSS Structure:

selector {
    property: value;
    property: value;
}

Real Example:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

Types of CSS

There are three ways to add CSS to HTML documents:

1. Inline CSS

CSS written directly in HTML elements using the style attribute:

<p style="color: red; font-size: 18px;">This is red text.</p>

2. Internal CSS

CSS written inside <style> tags in the HTML document's head:

<head>
    <style>
        p { color: blue; }
    </style>
</head>

3. External CSS

CSS written in separate .css files and linked to HTML:

<head>
    <link rel="stylesheet" href="styles.css">
</head>
