CSS Typography

Introduction to Typography

Typography is the art and technique of arranging text to make it readable, legible, and visually appealing. In web design, good typography enhances user experience, establishes hierarchy, and conveys your brand's personality.

Font Properties

1. Font Family

Specifies which font to use. Always provide fallbacks in case the preferred font isn't available.

Example: Font Families

/* Serif fonts - traditional, formal */
h1 { font-family: "Times New Roman", Times, serif; }

/* Sans-serif fonts - modern, clean */
p { font-family: Arial, Helvetica, sans-serif; }

/* Monospace fonts - code, technical */
code { font-family: "Courier New", Courier, monospace; }

/* Generic families */
.cursive { font-family: cursive; }
.fantasy { font-family: fantasy; }

Result:

Serif: "Times New Roman", Times, serif
Sans-serif: Arial, Helvetica, sans-serif
Monospace: "Courier New", Courier, monospace
Cursive: cursive
Fantasy: fantasy

2. Font Size

Controls the size of text using various units:

Example: Font Sizes

/* Absolute units */
.small { font-size: 12px; }
.medium { font-size: 16px; }
.large { font-size: 24px; }

/* Relative units */
.rem-size { font-size: 1.5rem; }    /* 1.5 × root font size */
.em-size { font-size: 1.2em; }      /* 1.2 × parent font size */
.percent { font-size: 120%; }       /* 120% of parent font size */

/* Viewport units */
.responsive { font-size: 4vw; }     /* 4% of viewport width */

Result:

12px - Small text for captions
16px - Standard body text
20px - Large body text
24px - Subheading size
32px - Heading size
48px - Large heading

3. Font Weight

Controls how bold or light the text appears:

Example: Font Weights

/* Numeric values (100-900) */
.thin { font-weight: 100; }
.light { font-weight: 300; }
.normal { font-weight: 400; }    /* Same as 'normal' */
.medium { font-weight: 500; }
.bold { font-weight: 700; }      /* Same as 'bold' */
.black { font-weight: 900; }

/* Keyword values */
.keyword-normal { font-weight: normal; }
.keyword-bold { font-weight: bold; }
.keyword-bolder { font-weight: bolder; }  /* Relative to parent */
.keyword-lighter { font-weight: lighter; }

Result:

100 - Thin/Ultra Light
300 - Light
400 - Normal/Regular
500 - Medium
700 - Bold
900 - Black/Ultra Bold

4. Font Style

Controls italic, oblique, or normal text styling:

Example: Font Styles

.normal { font-style: normal; }
.italic { font-style: italic; }
.oblique { font-style: oblique; }

Result:

Normal text style
Italic text style
Oblique text style

Text Properties

1. Text Alignment

Example: Text Alignment

.left { text-align: left; }
.center { text-align: center; }
.right { text-align: right; }
.justify { text-align: justify; }

Result:

Left aligned text - This is the default alignment for most languages that read left to right.
Center aligned text - Often used for headings, titles, or call-to-action elements.
Right aligned text - Commonly used for dates, prices, or in right-to-left languages.
Justified text - This text is stretched to align with both left and right margins, creating clean edges. However, it can sometimes create uneven spacing between words, so use it carefully.

2. Text Decoration

Example: Text Decoration

.none { text-decoration: none; }
.underline { text-decoration: underline; }
.overline { text-decoration: overline; }
.line-through { text-decoration: line-through; }

/* Multiple decorations */
.multiple { text-decoration: underline overline; }

/* Decoration with color and style */
.styled {
    text-decoration: underline;
    text-decoration-color: #E8BB17;
    text-decoration-style: wavy;
}

Result:

No decoration (normal text)
Underlined text
Overlined text
Line-through text
Multiple decorations
Styled decoration

3. Text Transform

Example: Text Transform

.none { text-transform: none; }
.uppercase { text-transform: uppercase; }
.lowercase { text-transform: lowercase; }
.capitalize { text-transform: capitalize; }

Result:

Original Text Case
uppercase text transform
LOWERCASE TEXT TRANSFORM
capitalize each word

Text Spacing

1. Line Height

Controls the space between lines of text:

Example: Line Height

.tight { line-height: 1; }        /* 100% of font size */
.normal { line-height: 1.2; }      /* 120% of font size */
.comfortable { line-height: 1.5; } /* 150% of font size */
.loose { line-height: 2; }         /* 200% of font size */

Result:

Line Height: 1 (Too tight)

This paragraph has very tight line spacing. The lines are too close together, making it difficult to read, especially for longer text content.

Line Height: 1.2 (Acceptable)

This paragraph has acceptable line spacing for headings but might still be too tight for body text in some cases.

Line Height: 1.5 (Recommended)

This paragraph has ideal line spacing for body text. It's comfortable to read and provides good visual separation between lines.

Line Height: 2 (Too loose)

This paragraph has very loose line spacing. While it's readable, the excessive space can make text feel disconnected.

2. Letter Spacing

Example: Letter Spacing

.tight { letter-spacing: -0.5px; }
.normal { letter-spacing: normal; }
.wide { letter-spacing: 1px; }
.wider { letter-spacing: 3px; }

Result:

Tight letter spacing
Normal letter spacing
Wide letter spacing
Wider letter spacing

3. Word Spacing

Example: Word Spacing

.normal { word-spacing: normal; }
.wide { word-spacing: 5px; }
.wider { word-spacing: 10px; }

Result:

Normal word spacing between words
Wide word spacing between words

Web Fonts

Use custom fonts from external sources like Google Fonts:

Using Google Fonts

<!-- In HTML head -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:wght@400;700&display=swap" rel="stylesheet">

/* In CSS */
.modern-sans {
    font-family: 'Inter', Arial, sans-serif;
}

.elegant-serif {
    font-family: 'Playfair Display', Georgia, serif;
}

.code-font {
    font-family: 'Source Code Pro', 'Courier New', monospace;
}

Result:

Playfair Display - Elegant Serif

Inter - Modern Sans-serif for body text

Source Code Pro - Code font

Typography Hierarchy

Create visual hierarchy using different font sizes, weights, and spacing:

Example: Typography Scale

h1 {
    font-size: 2.5rem;
    font-weight: 700;
    line-height: 1.2;
    margin-bottom: 1rem;
}

h2 {
    font-size: 2rem;
    font-weight: 600;
    line-height: 1.3;
    margin: 1.5rem 0 0.75rem 0;
}

h3 {
    font-size: 1.5rem;
    font-weight: 500;
    line-height: 1.4;
    margin: 1rem 0 0.5rem 0;
}

p {
    font-size: 1rem;
    font-weight: 400;
    line-height: 1.6;
    margin-bottom: 1rem;
}

.small-text {
    font-size: 0.875rem;
    color: #666;
    line-height: 1.5;
}

Result:

Main Heading (H1)

This is a paragraph that follows the main heading. It provides context and information in a readable format with appropriate line height and spacing.

Section Heading (H2)

Another paragraph that demonstrates the hierarchy. Notice how the different font sizes and weights create a clear visual order.

Subsection Heading (H3)

A shorter paragraph under the subsection heading.

Small text for captions, footnotes, or secondary information.

Font Pairing

Combine fonts effectively for better design:

Example: Serif + Sans-serif Pairing

/* Heading font - Serif for elegance */
h1, h2, h3 {
    font-family: 'Playfair Display', Georgia, serif;
    font-weight: 700;
}

/* Body font - Sans-serif for readability */
p, body {
    font-family: 'Inter', Arial, sans-serif;
    font-weight: 400;
    line-height: 1.6;
}

Result:

Elegant Serif Heading

This body text uses a clean sans-serif font that pairs beautifully with the serif heading. The contrast between the decorative serif and clean sans-serif creates visual interest while maintaining excellent readability.

Responsive Typography

Make text adapt to different screen sizes:

Example: Responsive Font Sizes

/* Fluid typography using clamp() */
h1 {
    font-size: clamp(1.5rem, 4vw, 3rem);
    /* minimum: 1.5rem, preferred: 4vw, maximum: 3rem */
}

/* Media queries for responsive text */
.responsive-text {
    font-size: 1rem;
}

@media (min-width: 768px) {
    .responsive-text {
        font-size: 1.125rem;
    }
}

@media (min-width: 1024px) {
    .responsive-text {
        font-size: 1.25rem;
    }
}

Result (resize window to see effect):

This text scales responsively with screen size

Readability Best Practices

Good vs Poor Readability

✅ Good Readability

This text demonstrates good readability practices. It uses a clean sans-serif font, appropriate font size (16px), comfortable line height (1.6), good contrast, and proper spacing. The text is easy to scan and comfortable to read for extended periods.

❌ Poor Readability

This text shows poor readability. It uses a decorative font that's hard to read, small font size, tight line height, poor contrast, and cramped letter spacing. Avoid these practices for body text.

Typography Best Practices

✅ Do:

  • Use a maximum of 2-3 fonts per design
  • Maintain consistent hierarchy with font sizes
  • Ensure good contrast (4.5:1 ratio minimum)
  • Use line-height of 1.4-1.6 for body text
  • Limit line length to 45-75 characters
  • Choose web-safe fonts with fallbacks
  • Test on different devices and screen sizes

❌ Don't:

  • Use too many fonts - it creates chaos
  • Use decorative fonts for body text
  • Make text too small (below 14px on mobile)
  • Use poor color contrast
  • Ignore mobile users
  • Stretch or distort fonts
  • Use all caps for long text
