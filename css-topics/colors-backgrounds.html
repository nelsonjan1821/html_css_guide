CSS Colors & Backgrounds

Introduction

Colors and backgrounds are fundamental to web design. They create visual hierarchy, convey meaning, establish brand identity, and enhance user experience. CSS provides multiple ways to specify colors and create stunning background effects.

CSS Color Values

CSS supports several ways to specify colors:

1. Color Names

CSS provides 147 predefined color names:

Example: Named Colors

h1 { color: red; }
p { color: blue; }
div { background-color: yellow; }

Result:

red
blue
green
orange
purple
brown

2. Hexadecimal Colors

Six-digit hex codes represent RGB values (#RRGGBB):

Example: Hex Colors

h1 { color: #0C2292; }      /* Our blue theme */
p { color: #E8BB17; }       /* Our gold theme */
div { background-color: #FFFFFF; } /* White */

Result:

#0C2292
#E8BB17
#FF0000
#00FF00
#0000FF
#000000

Short Hex Notation:

#F00 = #FF0000 (red)
#0F0 = #00FF00 (green)
#00F = #0000FF (blue)

3. RGB Colors

RGB values use red, green, and blue components (0-255):

Example: RGB Colors

h1 { color: rgb(12, 34, 146); }        /* Same as #0C2292 */
p { color: rgb(232, 187, 23); }        /* Same as #E8BB17 */
div { background-color: rgb(255, 0, 0); } /* Red */

Result:

rgb(12,34,146)
rgb(232,187,23)
rgb(255,0,0)
rgb(0,255,0)
rgb(0,0,255)
rgb(128,128,128)

4. RGBA Colors (with Transparency)

RGBA adds an alpha channel for transparency (0.0 to 1.0):

Example: RGBA Colors

.overlay { background-color: rgba(12, 34, 146, 0.8); }  /* 80% opaque */
.highlight { background-color: rgba(232, 187, 23, 0.3); } /* 30% opaque */

Result:

rgba(12, 34, 146, 0.9) - 90% Opaque
rgba(232, 187, 23, 0.7) - 70% Opaque
rgba(255, 255, 255, 0.5) - 50% Opaque

5. HSL Colors

HSL uses Hue (0-360°), Saturation (0-100%), and Lightness (0-100%):

Example: HSL Colors

h1 { color: hsl(225, 85%, 31%); }      /* Blue */
p { color: hsl(45, 90%, 50%); }        /* Gold */
div { background-color: hsl(0, 100%, 50%); } /* Red */

Result - HSL Color Wheel:

hsl(0,100%,50%)
hsl(60,100%,50%)
hsl(120,100%,50%)
hsl(180,100%,50%)
hsl(240,100%,50%)
hsl(300,100%,50%)

Background Properties

Background Color

Example: Background Colors

.blue-bg { background-color: #0C2292; }
.gold-bg { background-color: #E8BB17; }
.gradient-bg { background: linear-gradient(45deg, #0C2292, #E8BB17); }

Result:

Background Color: #0C2292
Background Color: #E8BB17

Linear Gradients

Example: Linear Gradients

/* Basic gradient */
.gradient1 { background: linear-gradient(to right, #0C2292, #E8BB17); }

/* Angled gradient */
.gradient2 { background: linear-gradient(45deg, #0C2292, #E8BB17); }

/* Multi-color gradient */
.gradient3 { background: linear-gradient(90deg, #0C2292, #E8BB17, #0C2292); }

Result:

Linear Gradient (to right)
Linear Gradient (45deg)
Multi-color Gradient

Radial Gradients

Example: Radial Gradients

/* Circular gradient */
.radial1 { background: radial-gradient(circle, #0C2292, #E8BB17); }

/* Elliptical gradient */
.radial2 { background: radial-gradient(ellipse, #E8BB17, #0C2292); }

Result:

Radial Gradient (circle)
Radial Gradient (ellipse)

Background Images

Example: Background Images

/* Background image */
.bg-image {
    background-image: url('pattern.png');
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}

Background Properties:

  • background-repeat: repeat, no-repeat, repeat-x, repeat-y
  • background-position: center, top, bottom, left, right, or specific values
  • background-size: cover, contain, or specific dimensions
  • background-attachment: scroll, fixed

Practical Color Tips

1. Color Accessibility

Good vs Bad Contrast:

2. Color Psychology

  • Blue (#0C2292): Trust, professionalism, stability
  • Gold (#E8BB17): Premium, excellence, achievement
  • Red: Urgency, importance, energy
  • Green: Success, growth, nature
  • Orange: Enthusiasm, creativity, warmth

Modern CSS Color Features

CSS Custom Properties (Variables)

Example: Color Variables

:root {
    --primary-blue: #0C2292;
    --accent-gold: #E8BB17;
    --text-white: #FFFFFF;
}

.button {
    background-color: var(--primary-blue);
    color: var(--text-white);
    border: 2px solid var(--accent-gold);
}

Best Practices

  • ✅ Maintain good contrast for accessibility (4.5:1 ratio minimum)
  • ✅ Use consistent color schemes throughout your design
  • ✅ Test colors on different devices and lighting conditions
  • ✅ Consider color blindness - don't rely on color alone for meaning
  • ✅ Use CSS variables for maintainable color systems
  • ✅ Keep your palette limited - 2-5 main colors work best
  • ❌ Avoid bright colors on large areas - they can cause eye strain
  • ❌ Don't use too many colors - it creates visual chaos
